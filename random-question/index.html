Random Question Console Quiz

Challenge from the Udemy JS course

Game has no UI, open console to play.

Coding instructions:

  • Build a function constructor called Question to describe a question. A question should include: a question, correct answer and all answers the player can choose from
  • Create a couple of questions & store them in an array
  • Select one random question and log it on the console, together with the possible answers (each question should have a number)
  • Use the 'prompt' function to ask the user for the correct answer. The user should input the number of the correct answer
  • Print to the console whether the answer is correct or not
  • After you display the result, display the next random question. Be careful to include the option to quit the game if the user writes 'exit' instead of the answer
  • Track the user's score and display it in the console
